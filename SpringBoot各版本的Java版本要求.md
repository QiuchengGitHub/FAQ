#<center>Spring Boot 各版本的Java版本要求</center>#
##概述
截至目前（20200520），官网上发布的springboot版本数已经高达260+，此文档只是选择市场占有率较高的版本来统计，毕竟在技术选型时肯定是选择稳定的大版本。

如果对所有springboot版本感兴趣的童鞋，请自行浏览[官网链接](https://docs.spring.io/spring-boot/docs/)
